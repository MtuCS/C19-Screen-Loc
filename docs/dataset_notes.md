# Dataset notes

## Classification
Dataset: COVID-19 Radiography Database

Original labels:
- COVID-19
- Normal
- Lung Opacity
- Viral Pneumonia

Binary mapping used in this work:
- COVID -> COVID
- Normal + Lung Opacity + Viral Pneumonia -> Non-COVID

## Localization
Dataset: SIIM-FISABIO-RSNA COVID-19 Detection

Used for lesion bounding-box localization.

## Important note
Do not upload original medical images to GitHub unless the dataset license explicitly permits redistribution in that form.
