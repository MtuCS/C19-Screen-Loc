# Reproducibility checklist

## Add before making the repository fully public

### Paper assets
- [ ] Final accepted PDF or arXiv/preprint PDF
- [ ] Final title and author list
- [ ] Correct citation entry
- [ ] Final venue, DOI, or EDAS metadata

### Code
- [ ] Stage 1 training code
- [ ] Stage 1 evaluation code
- [ ] Stage 2 training code
- [ ] Stage 2 evaluation code
- [ ] End-to-end inference pipeline
- [ ] Random seed control
- [ ] Configuration files

### Data
- [ ] Exact dataset version and download links
- [ ] Preprocessing instructions
- [ ] Patient-wise split description
- [ ] External test dataset preparation notes
- [ ] Label normalization rules

### Results
- [ ] ROC figure
- [ ] Confusion matrix
- [ ] Probability histogram
- [ ] YOLO training curves
- [ ] Qualitative localization examples
- [ ] Threshold analysis table

### Release quality
- [ ] Clear environment setup
- [ ] requirements.txt or environment.yml
- [ ] README with quick start
- [ ] License chosen
- [ ] Contact / issue policy
